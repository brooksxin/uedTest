技术摘要：GIS平台第三方接入文档

概述

功能说明

  • 系统集成了标点前台、标点信息展示、多边形区域展示、行政区域展示、导航五个底层接口，四个基本工具类（区域绘制搜索、公交导航、距离计算、周边搜索），客户信息展示、片区管理功能、签到功能三个典型demo样例，以及更底层方法。
  • 必要时需要引入jquery、jquery.ztree.js及ztree样式文件。

使用说明

  • 地图初始化

    • 允许配置参数，选择加载的工具类
    • body添加onload方法

    • AIMap_PARAMS参数说明：

      • MAPID：第三方系统定义的地图div容器ID（注意：需要定义好div的宽高，否则不显示。）

      • LOCATION：定位服务，该参数表示在地图初始化后以何种形式表示地图中心点的位置，共有四种形式，如下：

        1. 经纬度：如若想以传入的经纬度来初始化地图中心点，第三方需设置FLAG为true，并传入地图级别GRADE（3到18数字）以及经纬度。

        2. GPS定位：如若想通过GPS定位到地图中心点，第三方需设置FLAG为true，并传入GRADE和GPS经纬度坐标（数组格式）

        3. 中文关键字：如若想通过输入中文关键字来表示地图中心点，第三方需设置FLAG为true，并传入GRADE和中文地址

          特别注意：以上三种定位只能选择一种定位方式。

      • 工具类TOOLSBAR加载

        1. 设置FLAG为true

        2. 区域绘制搜索工具：设置AREADRAW为true，并传入AREADRAWURL（搜索的后台接口路径）

        3. 公交导航工具：设置DAOHANG为true

        4. 距离计算工具：设置JULI为true

        5. 周边搜索工具：设置ZHOUBIAN为true，并传入ZHOUBIANURL（周边搜索后台接口路径）

  1. <body onload="init();">
  2.     <div id="myMap"></div>
  3. </body>
  1. function init(){
  2.     var AIMap_PARAMS = {
  3.         MAPID: "myMap",
  4.         LOCATION: {     //地图定位服务，三种定位方式只能任选其一。
  5.             LNGLAT: {
  6.                 FLAG: true,
  7.                 GRADE: 13,
  8.                 POINTS:[{
  9.                     lng : "120.336658",
  10.                     lat : "30.279206"
  11.                 }]
  12.             },
  14.             GPS: {
  15.                 FLAG: true,
  16.                 GRADE: 13,
  17.                 POINTS: [{lng: "120.336658", lat: "30.279206"}]
  18.             },
  20.             ADDRESS: {
  21.                 FLAG: true,
  22.                 GRADE: 13,
  23.                 ADDR: "杭州"
  24.             }
  25.         },
  26.         TOOLSBAR: {//工具条
  27.             FLAG: true,
  28.             AREADRAW: true,//是否显示区域绘制搜索按钮
  29.             AREADRAWURL: "hz/sdkaction/common.action?uid=getMarkPointByArea",//区域绘制搜索url
  30.             CALLBACK:function(json){},//区域绘制搜索回调,参数为标记点信息
  31.             DAOHANG: true,//是否显示导航按钮
  32.             JULI: true,//是否显示距离按钮
  33.             ZHOUBIAN: true,//是否显示周边搜索按钮
  34.             ZHOUBIANURL: "hz/sdkaction/common.action?uid=getMarkPointBySearch"//周边搜索url
  35.         }
  36.     }
  37.     AIMap_loadScript(AIMap_PARAMS);
  38. }
  • 地图初始化完成后，回调AIMap_init方法，编写第三方自己代码，方法返回地图对象。

  2.     //地图初始化完成
  3.     myMap = map;
  4. }

  • 系统功能4：显示行政区域

    • 说明：在地图上显示省市行政区域，默认为杭州

    • 使用实例：

  1. var bdary = new AIMap.Boundary();
  2.     var value = prompt('请输入行政区域名：', '杭州')||"杭州";
  3.     bdary.get(value, function(rs){        //获取行政区域
  4.         AIMap_map.clearOverlays();        //清除地图覆盖物
  5.         var count = rs.boundaries.length; //行政区域的点有多少个
  6.         for(var i = 0; i < count; i++){
  7.             var ply = new AIMap.Polygon(rs.boundaries[i], {strokeWeight: 2, strokeColor: "#ff0000"}); //建立多边形覆盖物
  8.             AIMap_map.addOverlay(ply);  //添加覆盖物
  9.             AIMap_map.setViewport(ply.getPath());    //调整视野
  10.         }
  11.     })
