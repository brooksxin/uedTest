"css3的transition允许css的属性值在一定的时间内平滑的过度。这种效果可以在鼠标的点击、获得焦点、被点击或对元素的任何改变触发，并圆滑的以动画效果改变css的属性值。"

transition包括四个属性值：执行变换的属性：transition-property，变换持续的时间：transition-duration，在延续时间内的速率变化：transition-timing-function，变换延迟时间：transition-delay。

可改变的属性：color、数字、transform、visibility...
但并不是什么属性改变都为触发transition动作效果，比如页面的自适应宽度，当浏览器改变宽度时，并不会触发transition的效果。

transition-duration：用于指定元素转换过程的持续时间，单位为s(秒)或者ms(毫秒)。可以作用于所有元素，包括:before和:after伪元素。默认值为0。

transition-timing-function：变换类型
ease：逐渐变慢。(0.25, 0.1, 0.25, 1.0)
linear：匀速。(0.0, 0.0, 1.0, 1.0)
ease-in：加速。(0.42, 0, 1.0, 1.0)
ease-out：减速。(0, 0, 0.58, 1.0)
ease-in-out：加速然后减速，表现为刚开始变换慢，达到顶速后迅速变为0。(0.42, 0, 0.58, 1.0)
cubic-bezier：(x1, y1, x2, y2)

transition-delay：延迟改变。类似transition-duration。一般浏览器先解析后者。

transition: all .5s ease 1s; 属性值-duration-type-delay
